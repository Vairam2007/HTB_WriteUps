# 🧪 Lab Principles

<div align="center">

<table>
<tr>
<td align="center" width="280">
<h3>🔍 Evidence First</h3>
Every claim in a writeup must be supported by observable output, packet captures, logs, screenshots, or reproducible commands.
</td>
<td align="center" width="280">
<h3>🧠 Mechanism Over Tricks</h3>
The objective is understanding authentication, execution flow, permissions, memory, networking, and operating system behavior.
</td>
<td align="center" width="280">
<h3>♻️ Repeatable Process</h3>
Recon → Enumerate → Hypothesize → Test → Escalate → Document. The methodology matters more than the box itself.
</td>
</tr>
</table>

</div>
